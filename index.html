Quadro interativo que usa JavaScript para exibir 4 metas com cronômetros personalizados, calculando o tempo restante até cada objetivo.

A Diretora Mandou

Jogo de cartas interativas (estilo flashcards) que usam JavaScript e CSS para girar ao clicar, revelando comandos divertidos com emoji.
